# LLD Roadmaps

Use this order for interview prep:

- Start with the master template and practice explaining the flow aloud.
- Solve one offline game to learn clean entity boundaries.
- Solve one machine problem to learn state transitions.
- Solve one booking system to learn concurrency and payment flows.
- Revisit every problem in interview mode and reduce the answer to two minutes.

## Suggested Sequence

1. TicTacToe
2. Vending Machine
3. Parking Lot
4. BookMyShow
5. Splitwise
